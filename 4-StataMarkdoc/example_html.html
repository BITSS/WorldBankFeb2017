Conclusion

Writing a document with HTML in MarkDoc is so natural and does not require additional options or commands. MarkDoc automatically recognizes that you are writing with HTML. All that you need to do is to export a html file from the options. Suprisingly you can mix HTML with Markdown as long as you export the document in html format or any other document format that recognizes html such as docm.
